Pengalaman PKL

PKL di sekolah saya menjadi salah satu syarat untuk kelulusan. Kelas saya di berikan waktu selama 9 bulan untuk melaksanakan PKL. Saya PKL di PT. Multi Manao Indonesia, sebagai IT. Saya PKL bersama 3 teman saya, dan di berikan tugas untuk membuat sebuah aplikasi peminjaman barang. Saya di berikan tugas untuk menjadi Front End Developer dan Database Engineer.

Tugas dari Front End Developer yaitu, merancang ide dasar tampilan website atau aplikasi. Tugas dari Database Engineer yaitu, merancang, memelihara, dan mengoptimalkan infrastruktur data untuk pengumpulan, pengelolaan, transformasi, dan akses data.

Tidak hanya fokus pada sebuah aplikasi peminjaman barang saja, saya dan teman teman saya juga di berikan tugas mengentry sebuah data di Ms. Excel dan Spreadsheet, memperbaiki sebuah kabel jaringan, merakit kembali sebuah PC, memperbaiki PC, dan lain sebagainya.

Prestasi Juara 2 Web Desain Umsida

2 minggu sebelumnya, kami terpilih untuk mewakilkan sekolah dalam mengikuti perlombaan Web Desain yang di adakan oleh UMSIDA. Sekolah memberikan dukungan penuh untuk kami agar kami dapat mempersiapkan perlombaan. Selama 2 minggu kami menentukan tema website, menggambarkan sketsa desain dan mempersiapkan sebuah desain website untuk kami buat dalam perlombaan.

Saat hari perlombaan sudah tiba, sehingga kami sudah siap dengan desain yang sudah kami persiapkan di sekolah, di UMSIDA kami di sambut dan di berikan pengarahan di ruang aula UMSIDA.

Saat perlombaan kami di berikan batas waktu selama 3 jam untuk membuat sebuah desain website, setelah selesai membuat website, kami mempresentasikan website apa yang kami buat dan menjelaskan kepada para juri website yang sudah kami buat, para juri juga banyak memberikan saran yang bermanfaat agar kedepan nya kami dapat membuat website yang lebih bagus dan menarik.

Setelah mempresentasikan hasil website deisgn kami, kami di arahkan ke aula UMSIDA untuk ISHOMA dan setelah melaksanakan ISHOMA, panitia memberikan pengunguman juara dalam perlombaan. SMK Krian 1 mendapatkan Juara 2 & 3 Web Desain dan Juara 3 UI/UX Desain
